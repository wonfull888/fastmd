# fastmd.dev v0.2.0

## Release Focus

v0.2 is the Skill Release.

The product story is now clearer for agent-native usage: the skill name is `fastmd`, first use auto-generates a local token, and each publish returns one short URL that can be handed back to the user instead of a long inline report.

## What Changed

### 1. AI Integration section on homepage

- Added a dedicated homepage block explaining how the `fastmd` skill fits into agent workflows.
- Documented the push flow from report generation to returned URL.
- Highlighted the token bootstrap and ownership model.

### 2. README skill documentation

- Added explicit `fastmd` skill naming.
- Added expected first-run token behavior.
- Added a concise agent workflow description for shareable report delivery.
