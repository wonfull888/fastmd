# fastmd

**A Markdown fast-transfer pipeline for AI Agents and developers.**

Push Markdown from your terminal, get a shareable link instantly.

```bash
cat report.md | fastmd
# → https://fastmd.dev/x7y2
```

## Install

```bash
curl -fsSL https://fastmd.dev/install.sh | sh
```

## Quick Start

```bash
# Push
cat file.md | fastmd

# Get raw markdown (machine-readable)
curl https://fastmd.dev/x7y2.md

# Delete
fastmd delete x7y2
```
