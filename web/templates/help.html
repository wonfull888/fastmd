Help & FAQ

What is fastmd?

fastmd is a Markdown fast-transfer pipeline. Push any Markdown content from your terminal or AI agent and get a shareable link instantly — no account required.

How do I get my token?

Your token is auto-generated the first time you run any fastmd command. It's saved to ~/.config/fastmd/token.

I lost my token. Can I recover it?

No — tokens are one-way hashed. If you lose your token, you can no longer delete documents you created. Generate a new token by deleting ~/.config/fastmd/token and running any command again.

How large can a document be?

Maximum 1MB per document.

Do documents expire?

In v0.1, documents are stored permanently by default. TTL options (24h, 7d, 30d) are coming in a future version.

How do I read a document as raw Markdown?

Add .md to the URL: https://fastmd.dev/x7y2.md

Or send an Accept: text/plain header.

Can I use this with AI agents?

Yes — that's the primary use case. Call POST /v1/push from your agent with the Markdown content and token. The response gives you a short URL to share.
